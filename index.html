Big Data: geoJSON

The goal for this assignment is to simulate a data set that you may encounter during project week!

Let's focus only on transforming this collection. No need to invoke a constructor, or pass off control to Handlebars.

Start simple:
  • Transform the initial collection into an array of each object's properties.
    • We first want to pick up on:
    1. The zip code
    2. Its address (if appliable, otherwise 'undefined')
    3. Its neighborhood (same as address, but there should be only one neighborhood for each zip. Remove any additional neighborhoods beyond that - ex: "Greenwich Village, SoHo" becomes "Greenwich Village").
    4. Ensure that this new collection also contains the coordinates for each zip. We ideally want this "coordinates" property to be its own object with "lat" and "lng" properties since we'd like to incorporate Google Maps later.
  • Finally, populate a stats object to pick up on the top five neighborhoods associated with the most zips in Manhattan!
